Definition from IAI: Objectified relationship between an opening element and an building element that fills (or partially fills) the opening element. It is an one-to-one relationship.

NOTE view definitions or implementer agreements may restrict an opening to be filled by one filling element only.

First the door opening is created within the wall by IfcWall(StandardCase) o-- IfcRelVoidsElement --o IfcOpeningElement, then the door is inserted within the opening by IfcOpeningElement o-- IfcRelFillsElement --o IfcDoor.
