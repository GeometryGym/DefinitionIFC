The schema IfcSharedComponentElements provides the ability to represent different kinds of small parts, such as various accessories and fasteners. The schema contains the entity IfcElementComponent, a subtype of IfcElement from schema IfcProductExtensions, and relevant subtypes for fasteners, mechanical fasteners and discrete accessories along with their corresponding type objects. The schema also includes special subtypes of IfcFeatureElementSubtraction in order to provide the ability to represent building element edge forms as features.
