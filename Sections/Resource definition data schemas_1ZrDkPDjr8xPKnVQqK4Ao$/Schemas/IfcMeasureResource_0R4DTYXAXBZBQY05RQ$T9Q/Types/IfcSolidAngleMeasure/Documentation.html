Definition from ISO/CD 10303-41:1992: A solid angle measure is the value of an angle in a solid.

Usually measured in Steradians, (sr, m2/m2).

Type: REAL

NOTE Corresponding STEP name: solid_angle_measure, please refer to ISO/IS 10303-41:1994 for the final definition of the formal standard.
HISTORY New type in IFC Release 1.5.1.
