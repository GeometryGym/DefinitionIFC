Definition from IAI: The IfcConnectionPortGeometry is used to describe the geometric constraints that facilitate the physical connection of two objects at a port having a profile geometry (here IfcProfile). It is envisioned as a control that applies to the element connection relationships.

This entity defines the geometric location and configuration of a port on a distribution element. This information can be used to determine how to physically connect distribution elements.

HISTORY  New entity in IFC Release 2.0.
IFC2x Edition 2 CHANGE  The entity IfcConnectionPortGeometry is deprecated since Release IFC2x Edition 2 and shall no longer be used.
