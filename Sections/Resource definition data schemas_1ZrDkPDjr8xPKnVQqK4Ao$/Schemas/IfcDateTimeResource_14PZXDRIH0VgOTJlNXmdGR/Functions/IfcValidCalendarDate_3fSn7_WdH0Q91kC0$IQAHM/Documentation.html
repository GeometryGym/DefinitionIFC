Definition from ISO/CD 10303-41:1992: Determines whether the components of a calendar date (IfcCalendarDate) indicate a valid date. If the calendar date is valid, the function returns TRUE, otherwise it returns FALSE.

NOTE Corresponding STEP name: valid_calendar_date, please refer to ISO/IS 10303-41:1994 for the final definition of the formal standard.
HISTORY New function in IFC Release 1.5.1.
